# effectsurf: Interactive 3D Estimated Marginal Surfaces for Conditional Effect Visualisation

Extends traditional 2D estimated marginal means (EMMs) into interactive
3D Estimated Marginal Surfaces (EMS). Provides model-agnostic
visualisation of predictions, conditional effect sizes, and conditional
average treatment effects (CATE) from fitted statistical models.
Surfaces are rendered as interactive 'plotly' 3D charts with categorical
stratification, confidence envelopes, 2D profile projections, and
sensitivity analysis. Supports 100+ model classes via 'marginaleffects',
with optional 'gratia' integration for GAM smooth surfaces.
